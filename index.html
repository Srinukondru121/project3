html
html is a markup language
  • it was introduced in 1991
  • it was discovered by Tim Berners Lee
  • HTML describes the structure of a Web page
  • HTML consists of a series of elements
heading tags
These provide heading styles for the text.
  1. h1 represents most important heading
  2. h2 represents second important heading
  3. h3 represents third important heading
  4. h4 represents fourth important heading
  5. h5 represents fifth important heading
  6. h6 represents least most important heading
paragraph tags
A paragraph always starts on a new line, and is usually a block of text.
  1. the hr tag defines a thematic break in an HTML page, and is most often displayed as a horizontal rule.
  2. the HTML br element defines a line break.
  3. The HTML pre element defines preformatted text.
formatting tags
HTML contains several elements for defining text with a special meaning.
  • b- bold text
  • del- deleted text
  • i- italic text
types of tags
these specify difference between tags which have have both open and closing tag and tags which have only open tags
  • there are 2 types of tags
    1.paired/container tag
    2.void tag
  • paired tag defines that the element has both open tag and closing tag
  • void tag defines that the element have only open tag but not closing tag
block and inline elements
these element has a default display value, depending on what type of element it is.
  1. a block-level element always starts on a new line, and the browsers automatically add some space
  2. a block-level element always takes up the full width available
  3. an inline element does not start on a new line.
  4. an inline element only takes up as much width as necessary
table tag
HTML tables allow web developers to arrange data into rows and columns.
  1. A table in HTML consists of table cells inside rows and columns.
  2. Each table cell is defined by a td and a /td tag.
  3. A table cell can contain all sorts of HTML elements: text, images, lists, links, other tables, etc.
image tag
Images can improve the design and the appearance of a web page.
  • Images are not technically inserted into a web page; images are linked to web pages.
  • The img tag is empty, it contains attributes only, and does not have a closing tag.
  • The img tag has two required attributes:
      • src - Specifies the path to the image alt - Specifies an alternate text for the image
anchor tag
The a tag defines a hyperlink, which is used to link from one page to another.
  1. An unvisited link is underlined and blue
  2. A visited link is underlined and purple
  3. An active link is underlined and red
url structure
A URL is another word for a web address.
  • URLs can only be sent over the Internet using the ASCII character-set.
  • URL encoding converts non-ASCII characters into a format that can be transmitted over the Internet.
  • URL encoding replaces non-ASCII characters with a "%" followed by hexadecimal digits.
list tags
lists allow web developers to group a set of related items in lists.
  1. an unordered list starts with the ul tag. Each list item starts with the li tag.
  2. an ordered list starts with the ol tag. Each list item starts with the li tag.
  3. A description list is a list of terms, with a description of each term. The dl tag defines the description list, the dt tag defines the term (name), and the dd tag
